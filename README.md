# Backbone to RWKV-7 Skill

Install this skill into the current project's `.codex/skills` directory.

## macOS, Linux, or Git Bash

Run from the target project root:

```sh
curl -fsSL https://raw.githubusercontent.com/Jellyfish042/backbone2rwkv_skill/main/install.sh | sh
```

Or you can install the English version:

```sh
curl -fsSL https://raw.githubusercontent.com/Jellyfish042/backbone2rwkv_skill/main/install.sh | BACKBONE2RWKV_LANG=en sh
```

## Windows PowerShell

Run from the target project root:

```powershell
iex "& { $(irm https://raw.githubusercontent.com/Jellyfish042/backbone2rwkv_skill/main/install.ps1) } -Lang zh"
```

Or you can install the English version:

```powershell
iex "& { $(irm https://raw.githubusercontent.com/Jellyfish042/backbone2rwkv_skill/main/install.ps1) } -Lang en"
```
